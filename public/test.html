Yatzy Dice Roller

Rules

You have up to 3 rolls per turn. In each turn you can choose which dice you keep (the rest are rerolled). During each turn you must choose a method to score as described on the scorecard. Each scoring method must be used only once per game.

Controls

  • Click the roll button to roll
  • Click which dice you want to change the re-roll state. A red outline indicates that it is a keeper.
  • Click the scoring method on the scorecard to finish turn and choose scoring method
  • Click the new game button to start a new game
